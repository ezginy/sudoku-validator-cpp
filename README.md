# Sudoku Validator (C++)

A C++ program that validates Sudoku solutions using file handling and multidimensional arrays.

## Features
- Reads Sudoku data from a text file
- Validates rows
- Validates columns
- Validates 3x3 subgrids
- Detects missing or extra numbers in the file
- Reports invalid rows, columns, and subgrids

## Technologies
- C++
- File I/O
- Multidimensional Arrays

## File Format
The program reads Sudoku data from a `sudoku.txt` file.

Example:

5 3 4 6 7 8 9 1 2  
6 7 2 1 9 5 3 4 8  
1 9 8 3 4 2 5 6 7

## Output
- `Valid Sudoku`
- `Invalid Sudoku`
